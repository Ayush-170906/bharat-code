## What is this?

**Bharat Code** is an AI-powered code assistant built specifically for Indian developers. It uses Claude AI to help you understand, debug, convert, and review code — and explains everything in your preferred Indian language.

## Features

| Feature | Description |
|---|---|
| **Explain Code** | Understand what any code does, explained clearly |
| **Debug & Fix** | Find bugs and get a fixed version instantly |
| **Convert Language** | Rewrite code in Python, JavaScript, Java, C++, Go, Rust, etc. |
| **Code Review** | Get expert feedback on quality, security, and best practices |

### Supported Languages
🇮🇳 English · हिन्दी · मराठी · தமிழ் · తెలుగు · বাংলা · ಕನ್ನಡ

## Tech Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript (zero dependencies)
- **AI:** Claude Sonnet (Anthropic API)
- **Deployment:** Netlify

## Deploy in 60 Seconds

1. Fork this repo
2. Go to [netlify.com](https://netlify.com) → New site from Git
3. Connect your forked repo
4. Deploy — no build command needed!

## Run Locally

```bash
git clone https://github.com/Ayush-170906/bharat-code
cd bharat-code
# Open index.html in your browser — that's it!
```

## How to Use

1. Select a **mode** (Explain / Debug / Convert / Review)
2. Choose your **output language**
3. Paste your code
4. Press **Run** or `Ctrl+Enter`
